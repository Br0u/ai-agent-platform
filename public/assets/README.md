# 正式素材目录

上线前需要补齐：

- Logo：SVG 与透明 PNG，浅色/深色背景各一版
- 产品真实截图：主页、Agent Studio、Workflow、Knowledge Base 等
- 产品架构图
- 客户案例 Logo 及公开授权
- 品牌字体文件或可合法使用的字体清单

在素材缺失期间，设计稿只使用带文字说明的占位区域。

## 占位规则

- 占位资产集中定义，页面不得各自创建不同版本。
- 保持与未来真实资产一致的宽高比，避免替换后布局重做。
- 不伪造Logo、客户名称、电话号码、备案号、产品截图和业务数据。
- 演示数据必须明确标记为演示用途。
- 正式上线前执行一次全站占位扫描，未替换项阻止生产发布。
