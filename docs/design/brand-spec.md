## 色彩系统

- Blue / Primary: `#3157D5`
- Indigo / Structural: `#4F46E5`
- Violet / Selective accent: `#7C3AED`
- Ink: `#111827`
- Muted ink: `#5B6475`
- Canvas: `#F5F7FC`
- Surface: `#FFFFFF`
- Dark canvas: `#0C1020`

使用规则：蓝色承担主操作，靛色承担结构和选中态，紫色只用于强调和状态识别；不使用大面积蓝紫渐变背景。
